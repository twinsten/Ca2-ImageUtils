# About CellRex

## Project Overview

CellRex is a specialized tool for organizing and enriching biological data files with metadata. It streamlines the process of documenting experimental conditions and managing research data.

## Key Features

- **File Organization**: Automated sorting and structuring of biological data files
- **Metadata Enrichment**: Comprehensive parameter documentation
- **Influence Group Management**: Flexible organization of experimental conditions
- **Quality Control**: Built-in validation for data consistency
