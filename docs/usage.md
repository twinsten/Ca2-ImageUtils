# Usage Guide

## Overview
CellRex helps you organize and enrich biological data files with metadata. This guide covers the essential workflows and features of the application.

## Basic Workflow

1. **File Selection**
   - Launch CellRex
   - Select your biological data files
   - Choose the target directory for processed files

2. **Parameter Configuration**
   - Set biological parameters (Species, Origin, Cell Type)
   - Configure experimental settings
   - Add documentation metadata

3. **File Processing**
   - Review your settings
   - Process files
   - Verify output

## Working with Influence Groups

Influence groups help organize experimental conditions in multiwell plates:

- **Single Influence Group**: Use when all wells share similar conditions
- **Multiple Influence Groups**: Use when tracking different variables
  - Example: IG1 for treatment + disease, IG2 for disease only

## Best Practices

1. **File Organization**
   - Use consistent naming conventions
   - Group related files together
   - Maintain clear experiment boundaries

2. **Metadata Management**
   - Fill all required fields
   - Use standardized terminology
   - Include relevant keywords

3. **Quality Control**
   - Verify parameter selections
   - Double-check influence group assignments
   - Review processed files

## Troubleshooting

Common issues and their solutions:

| Issue | Solution |
|-------|----------|
| File not processing | Verify file format and permissions |
| Missing metadata | Check required fields |
| Incorrect grouping | Review influence group settings |
