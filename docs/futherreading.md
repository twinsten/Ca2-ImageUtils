# Further Reading: Ca2+ Imaging and Neurofinder Datasets

## Background on Calcium Imaging

Calcium imaging is a powerful technique used in neuroscience to visualize and analyze neuronal activity. By using fluorescent calcium indicators, researchers can observe changes in neuronal activity in real-time.

### Key Concepts

#### Calcium Indicators
- Fluorescent molecules that change brightness when binding to calcium ions
- Allow visualization of neuronal activation through calcium concentration changes
- Common indicators include:
  - GCaMP family
  - Fluo-4
  - Oregon Green BAPTA

#### Region of Interest (ROI) Detection
The project's core functionality revolves around detecting and comparing Regions of Interest (ROIs) in calcium imaging datasets.
